Services

$history

Methods:
go(relativeIndex)
Goes back/forward in history
Properties:
urlStack
Array of objects, one for every entry in the history. Every entry has a url property.
activeIndex
The index in urlStack of the current history index
previousIndex
The previous index in urlStack. Useful to detect the browser back button.

jqmViewCache

This is a cache created by angular's cacheFactory and stores all cached views. It can be used to evict cached views on demand. For the api see cacheFactory.
